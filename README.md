# personal-deep-profile

> 个人全维度探索档案生成器 · Claude Code Agent Skill
> **一句出生信息进，四份个人档案出。**

## 干什么用

输入「日期 + 时间 + 地点 + 性别」（例：`1995-08-20 14:30 杭州 女`），自动产出：

1. **个人命理综合分析报告** — 八字 / 紫微 / 西洋占星 / 灵数 / 玛雅 / 人类图等多体系交叉分析，只把 ≥3 体系互证的结论写为高置信
2. **行动规划** — 当年逐月行动表 + 未来 2–3 年关键窗口 + 复盘节点
3. **命盘参数底稿** — 全部盘面数据存档，可审计
4. **通俗版人物总结与行动指南** — 全文零术语的大白话版本，家人朋友也能直接看懂
5. **测评×命理交叉分析**（可选）— MBTI / HBDI / 霍兰德报告与命盘逐条对撞
6. **memory 沉淀** — 盘主档案永久入库，后续问流年/合盘零成本召回

## 输入降级阶梯

| 给什么 | 得到什么 |
|---|---|
| 只给一句出生信息 | 自排盘（双源核验）→ 3 份报告 |
| + 排盘截图文件夹 | 截图转录 × 自排盘互校（最高置信，且解锁七政四余/二十八宿） |
| + 测评 URL 或口述类型 | 追加测评交叉报告 |

## 质量保险

- 八字：`sxtwl` × `lunar_python` 双库对算，干支不一致即停
- 占星：瑞士星历表 + 独立天文公式核对（`scripts/verify_ascendant.py`，误差 >0.1° 停）
- 任何体系未过双源核验 → 砍掉并注明，不输出未核验的盘
- 出生信息多源矛盾 → 停下问用户，不猜
- 3 轮自检循环未过 → 写日志标 TODO，绝不静默糊弄
